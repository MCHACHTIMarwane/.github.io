Python

c'est quoi python ?

Python est un langage de programmation généraliste, c’est à dire qu’il peut-être utilisé pour construire une grande variété d’applications. C’est un très bon langage de programmation pour débuter car il est lisible et il possède des structures simple à comprendre
Python est un langage de programmation interprété. Cela signifie qu’un programme écrit en Python n’as pas besoin d’être compilé à l’avance pour fonctionner. Ce système permet aux programmeurs de tester rapidement des morceaux de code. Ce qui augmente significativement la vitesse de développement

Algorithmes de recherche:

Un algorithme de recherche est un algorithme qui permet de rechercher un elément dans une liste de différentes facons. Par exemple il ya les algorithmes de recherche dichotomique ou encore les algorithme dit de recherche "naif".

Algorithme de tri par insertion:

Un algorithme de tri est un algorithme qui permet d'organiser une collection d'objets selon une relation d'ordre déterminée. Les algorithmes de tri éxecutent un processus classement d'un ensemble d'éléments dans un ordre donné. Par exemple, trier N entiers dans l'ordre croissant, ou N noms dans l'ordre alphabétique. Tout ensemble muni d'un ordre total peut fournir une suite d'éléments à trier.

il en existe plusieurs types :

  • le tri par selection
  • Le tri par insertion
  • Le tri contraint
